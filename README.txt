# Our Little Pack

A cute animated dog showcase website made for my 3 dogs.

This website has two main purposes:

1. To showcase my dogs and their personalities
2. To provide a QR-code-accessible page in case any of them get lost

Each dog has their own profile page, so a QR code on their collar can link directly to their personal page.

## Features

- Animated homepage
- Dog profile pages
- Gallery page
- Contact page
- Glow cursor effect
- Top scroll progress bar
- Reveal animations
- Anime.js motion effects
- Mobile-friendly responsive design
